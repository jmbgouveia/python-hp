# 💥 Suno WAV to MP4 Premium Remuxer v3.1

Interface gráfica (GUI) ultra-leve em Python para fundir o áudio Lossless do Suno AI com a imagem de capa (Art) num vídeo MP4 perfeitamente otimizado para redes sociais.

## ✨ Características de Elite
* **Modo Social (AAC 320kbps):** O teto máximo de áudio tolerado pelo Facebook/Instagram, processado com um filtro dinâmico que corrige dimensões de imagens ímpares (evita os bugs de renderização do Windows/Clipchamp e reduz ficheiros de 400MB para 19MB).
* **Modo Cofre (Lossless Copy):** Injeta o fluxo binário do teu WAV original direto no contentor de vídeo com ZERO compressão. Ideal para arquivo de estúdio.
* **Interface Dark Mode:** Desenvolvido em Tkinter com o tema `clam` costumizado.

## 🚀 Como Usar
1. Faz o download do script `SunoRemuxer.py`.
2. Garante que tens o `ffmpeg.exe` na mesma pasta do script.
3. Executa o programa: `python SunoRemuxer.py`
